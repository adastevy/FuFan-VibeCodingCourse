# Style · 微博

## When to Use
- Writing a Weibo post or thread opener
- Distilling a longer piece into a punchy microblog
- NOT for 公众号 articles or 小红书 notes

## Voice & Length
- Tone: direct, opinionated, newsy — no preamble
- Hard limit: **≤280 characters** (target ≤140 full-width CJK chars)
- First sentence IS the point; zero warm-up allowed

## Structure

Option A — Single punch:
```
[核心观点]。[一句支撑]。#话题#
```

Option B — Short list (2–3 items max):
```
[引子]：
· [点1]
· [点2]
#话题#
```

## Tags & Mentions
- 1–2 hashtags only: `#AI #Claude4` — more dilutes reach
- @mention sparingly: only when directly relevant to a specific account

## Common Mistakes
| Excuse | Reality |
|--------|---------|
| 把公众号摘要搬过来 | 超字数，读者看不完，系统截断 |
| 无 hashtag | 发现率接近零，算法无法归类 |
| 铺垫太长 | 微博读者1秒判断，开头即结论 |
| 超过2个话题标签 | 显得 spam，降低互动率 |
